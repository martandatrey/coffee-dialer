# Coffee Dialing Calculator

A single-page interactive "Coffee Dialing Calculator" built with React, Tailwind CSS, and Lucide-React.

## Features
- **Reactive Input Sliders**: Adjust Dose, Water, Grind, Temp, and Time.
- **Golden Ratio Lock**: Automatically adjusts Water based on Dose (1:16.6 ratio).
- **Taste Profiler**: Auto-adjusts parameters based on "Sour" or "Bitter" feedback.
- **Coffee Shop Aesthetic**: Warm, premium design.

## How to Run

1.  **Install Dependencies** (if you haven't already):
    ```bash
    npm install
    ```

2.  **Start the Development Server**:
    ```bash
    npm run dev
    ```

3.  **Open in Browser**:
    Click the link shown in the terminal (usually `http://localhost:5173`).

## Build for Production

To create a production build:
```bash
npm run build
```
The output will be in the `dist` directory.
